  3. Dirección

    Se puede definir la dirección como horizontal o vertical. '.flex-row' o '.flex-row-reverse' para dirección horizontal y '.flex-column' o '.flex-column-reverse' para dirección vertical.

  7. Alineación

    Use align-items utilities on flexbox containers to change the alignment of flex items on the cross axis (the y-axis to start, x-axis if flex-direction: column). Choose from start, end, center, baseline, or stretch (browser default). La utilidad '.align-items' en contenedores flexbox se usa para cambiar la alineación horizontal o vertical. Se puede alinear un elemento con '.align-self'
